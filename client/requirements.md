## Packages
framer-motion | Required for smooth, high-quality E-sports animations and pulsing effects
react-hook-form | Required for handling the subscriber newsletter form
@hookform/resolvers | Required for Zod validation on the frontend
lucide-react | Required for gaming-themed icons
clsx | Required for class merging
tailwind-merge | Required for dynamic class handling

## Notes
- The application uses a Neon Orange (#f39c12) and Dark Grey/Black theme to match the E-sports requirement.
- The `subscribers` API is integrated into a newsletter section at the bottom to capture emails.
- Animations and glowing effects are achieved via custom CSS and framer-motion.
